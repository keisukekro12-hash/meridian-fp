# よく使うパターン

> ユーザーがよく頼む作業の型を記録する。パターンが蓄積されると秘書の段取りが速くなる。

## フォーマット

```
### パターン名
- **トリガー**：どんな依頼のとき
- **手順**：どう進めるか
- **注意点**：気をつけること
```

---

### Threads投稿リアクションテンプレートを出す
- **トリガー**：「投稿のリアクションテンプレートを出して」
- **手順**：`Threads運用/テンプレート/リアクション記録.md` を読み込み、テンプレート部分をそのまま出力する
- **注意点**：ファイルの内容をそのままコピペできる形で出すこと。説明は最小限に
